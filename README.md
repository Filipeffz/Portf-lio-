# 🚀 Portfólio - Filipe Alves 

Bem-vindo ao meu portfólio profissional! Este projeto foi desenvolvido para apresentar minhas habilidades, projetos, experiências e formas de contato de maneira moderna e responsiva.

## 📋 Sobre o Projeto

Este portfólio foi criado com o objetivo de reunir minhas principais informações profissionais em um único lugar, permitindo que recrutadores, empresas e colegas conheçam melhor meu trabalho e minha trajetória na área de tecnologia.

## ✨ Funcionalidades

- Página inicial com apresentação profissional
- Seção sobre mim
- Exibição de projetos desenvolvidos
- Links para GitHub e LinkedIn
- Formulário de contato
- Layout responsivo para dispositivos móveis e desktops
- Interface moderna e intuitiva

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- React
- TypeScript
- Vite
- Tailwind CSS
